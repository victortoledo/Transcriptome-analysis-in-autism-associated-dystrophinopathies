# Transcriptome-analysis-in-autism-associated-dystrophinopathies
Bulk RNA-seq of 18 iPSC-derived samples from 9 males: 3 controls, 3 DMD/BMD without ASD, 3 DMD/BMD with ASD. Neural stem cells (day 30) and neurons (day 60) analyzed per individual. Pairwise differential expression compared the three groups separately for each neural stage.
 1. Alignment with STAR
 2. Quantification with RSEM
 3. Preprocessing
 4. Differential expression analysis
